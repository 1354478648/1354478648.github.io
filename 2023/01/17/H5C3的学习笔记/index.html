H5C3的学习笔记

发表于 2023-01-17 更新于 2023-10-10 分类于 前端 阅读次数：
本文字数： 10k 阅读时长 ≈ 19 分钟

HTML

基本标签

  • <h1>###</h1>：标题
  • <b>###</b>：字体加粗
  • <i>###</i>：字体倾斜
  • <br/>：换行
  • <hr/>：分隔线
  • <p>###</p>：正文

文件链接&图片&视频

  • <a href="###">###</a>：链接
    href可以是外部链接，也可以是本地的，当输入外部链接的时候，需确认网址完整无误。当输入本地链接时，需要需确认目标文件和当前文件的文件路径，如果同属于相同文件夹下，可以直接输入文件名；如果属于不同文件夹下，需要输入文件路径（退级写法：“../../###.html”）
  • <img src="###" width="###">：嵌入图片。
    src是目标文件路径，写法和href类似
    width（或height）可以设置图片的大小，只要不同时设置width和height，可以使图片等比例缩放
  • <video src="###" controls width="###">###</video>：嵌入视频
    controls属性是视频控制器，只有写上该属性才能使视频进行播放暂停等操作

列表

  • <ul><li>###</li></ul>：无序列表
  • <ol><li>###</li></ol>：有序列表。可以通过type属性更改序号，默认是数字
  • <table></table>：表格
  • <tr></tr>：列
  • <td>###</td>：表中数据

容器

  • <span></span>：无法换行
  • <div></div>：独占一行

输入

  • text：普通的文本输入框。placeholder属性可作为提示展示给用户
  • password：密码输入框。用户输入的数据是不可视的
  • date：选择日期
  • range：鼠标可拖动的滑块
  • file：上传文件
  • checkbox：复选框
  • <textarea>###</textarea>：可以输入很多文本

meta标签

  • charset="UTF-8：定义字符集
  • name="###" content="###"：定义页面属性
    description：页面介绍；author：页面作者；keywords：页面关键字；viewpoint：页面大小

CSS

颜色color

  • color：颜色，可以输入三种形式的参数：颜色名、rgb、十六进制数
  • background-color：背景颜色
  • border：边框，可以选择粗细程度、实线（或虚线）、颜色等

内边距padding&外边距margin

  • padding：内边距，参数决定着文字到边框border的距离
  • margin：外边距，参数决定着边框border到上下两个容器（块）的距离
    如果参数只有一个，那么该参数决定上下左右四个方向的距离；一旦有四个参数，将从上右下左的顺序决定距离；也可以设置“###-top、###-right、###-bottom、###-left“的形式，单独设置距离

浮动float

  • float：参数是浮动的方向，通常用于图片元素，使其和文字并列显示在同一行

块元素类型display

  • display：参数有inline、block、两者相互混合
    在本例中，<p>标签的块元素属性是整行的，设置为inline后就不再独占一行；<span>标签设置为block后就独占一行了

定位position

  • position：参数有fixed、relative、absolute等，默认值是static
    fixed：固定定位，它使容器相对于窗口进行偏移，通过设置top、bottom、left、right的参数来决定偏移距离
    relative：相对定位，它使容器相对于原来应该在的位置进行偏移（如果不加任何top、bottom、left、right的情况下，和static正常定位是一样的）
    absolute：绝对定位，它使容器相对于祖先元素的位置进行偏移

透明度opacity

  • opacity：透明度，参数范围为0~1

圆角border-radius

  • border-radius：圆角，如果设置的参数是容器长宽的一半，那么将会就是圆形

style标签

  • style：在<style>标签里设置某标签的样式属性，可以使所有该标签改变样式
    通过新建后缀名为.css的文件，可以将所有样式属性保存在该文件内。然后在<head>标签里引入外部css档案<link rel="stylesheet" href="###.css"/>，使样式属性作用在.html文件里

分类class&id

  • 通过分类，使样式属性单独地作用在某一分类中。一个标签身上的分类可以有多种，参考本例中的麻婆豆腐，在css文件中通过“.###”为分类附上样式属性
  • id和class类似，但id的样式属性只能使用一次，而且在css文件中通过“#”附上样式属性

css选择器

  • 如果多个标签（或分类等）的样式属性是一样的，则可以写在一起，用逗号分隔
  • 星号可以将样式属性作用在整个页面中
  • 如果要指定某一标签里的标签的样式属性，那么需要指定标签的路径
  • 标签内的属性也可以被附上样式属性，在css中使用一对中括号并填上属性名，则可以将样式属性作用在任何有该属性的标签上
  • 当设置了hover，当鼠标滑动到该元素时，样式属性会作用在该元素上

flex属性

  • flex属性：让所有块都有对应比例的长度
  • order属性：以参数的大小决定块的排序
  • flex-wrap: wrap：使块能够在位置不够的时候换行
  • flex-direction: column：使块从上往下排列，参数还可以填写row、row-reverse、column-reverse
  • justify-content: center：使块在主轴（由flex-direction决定，默认是row）居中，其他参数详见https://www.runoob.com/cssref/css3-pr-justify-content.html
  • align-items: center：使块在侧轴居中，其他参数详见https://www.runoob.com/cssref/css3-pr-align-items.html
  • align-content: center：使块在侧轴居中，和align-items: center不同的是，该属性在块换行的情况才有用

动画animation

  • 通过定义@keyframes 动画名来编辑动画效果，然后在想要实现动画效果的样式属性内添加animation-name: 对应动画名;
  • animation-duration: #s;：单次动画的持续时间
  • animation-iteration-count:infinite;：使动画无限重复，参数也可以是数字
  • animation-delay: #s;：设置动画在用户打开页面后#秒后开始动画
  • 通过设置%的形式实现的彩虹效果
