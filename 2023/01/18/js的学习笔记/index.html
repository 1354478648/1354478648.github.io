变量定义

JS是一门弱类型语言，变量可以存放不同类型的值
var是用来定义全局变量的，如果只想在代码块里暂时地使用某个变量，可以使用let关键字来定义变量，且不允许重复定义同名的变量

const关键字用于声明只读变量。一旦声明，常量的值就不能改变

数据类型

在JS中分为：原始类型和引用类型

原始类型 含义 备注
number 数字 整数、小数、NaN
string 字符串 单双引号都可
boolean 布尔 true false
null 对象为空 null的数据类型是object，引用类型也是object
undefined 未定义 当声明的变量未初始化，该变量的默认值是undefined

运算符

类型转换：

  • 其他类型转为number
  1. string：按照字符串中的字面值进行转换，如果不是数字，则转为NaN。一般使用parseInt()进行手动转换
  2. boolean：true转为1，false转为0
  • 其他类型转为boolean
  1. number：0和NaN转为false，其他数字转为true
  2. string：空字符串转为false，有内容的字符串转为true
  3. null：只会转为false
  4. undefined：只会转为false

BOM

浏览器对象模型（Browser Object Model），JS将浏览器的各个组成部分封装为对象
1. Window对象
浏览器窗口对象，调用对象的时候直接使用即可，而且window.可以省略

  • 属性：用于获取其他BOM对象（history,Navigator,Screen,location）
  • 方法：
    alert()：显示带有一段消息和一个确认按钮的警告框
    confirm()：显示带有一段消息以及确认按钮和取消按钮的对话框

setInterval(function,毫秒值)：按照指定的周期（ms）来调用函数或计算表达式，周期性地执行
setTimeout(function,毫秒值)：在指定的毫秒数后调用函数或计算表达式，只执行一次

2. History对象
获取History对象可以window.history，其中window.可以省略

  • 方法：
    back()：加载history列表中的前一个URL
    forward()：加载history列表中的后一个URL

3. Location对象
获取Location对象可以window.location，其中window.可以省略

  • 属性：
    href()：设置或返回完整的URL

DOM

文档对象模型（Document Object Model ）
1. Element元素对象
通过使用Document对象的方法Document.###来获取

  • getElementById：根据id属性值获取，返回一个Element对象
  • getElementByTagName：根据标签名称获取，返回Element对象数组
  • getElementByName：根据name属性值获取，返回Element对象数组
  • getElementByClassName：根据class属性值获取，返回Element对象数组

常见Element对象的使用：

  1. img对象（src属性）
  1. div对象（style属性，innerHTML属性）
    div对象没有自己独特的属性或方法，完全继承于Element对象
  1. input对象（checked属性）

事件监听

JS可以在事件被侦测到时执行代码
事件绑定：
方式一：通过HTML标签中的事件属性进行绑定

方式二：通过DOM元素属性绑定

常见事件：

  • onblur：失去焦点，例如用户在文本框中输入内容后在文本框外点击鼠标
  • onfocus：获得焦点，例如用户点击文本框
  • onchange：内容被改变，例如文本框中更改内人
  • onclick：内容被点击，例如复选框被点击
  • onkeydown：按键被按下，例如用户无法在文本框中输入数字
  • onmousemove：鼠标被移动
  • onmouseover：鼠标移到某元素上
  • onsubmit：点击确认按钮，返回值是布尔，返回true时表单会被提交

正则表达式

正则表达式定义了字符串组成的规则
方式一：直接量（常用）（不要加引号，否则是字符串）

方式二：创建RegExp对象

语法：

  • ^：表示开始
  • $：表示结束
  • [ ]：表示某个范围内的单个字符，比如[0-9]单个数字字符
  • .：表示任意单个字符，除了换行和行结束符
  • \w：代表单词字符，字母、数字、下划线，相当于[A-Za-z0-9]
  • \d：代表数字字符，相当于[0-9]

量词：

  • +：至少一个
  • *：零个或多个
  • ？：零个或一个
  • {x}：x个
  • {m,}：至少m个
  • {m,n}：至少m个，最多n个

reg对象有个方法test()，用于判断指定字符串是否符合规则，返回布尔值

Vue

Vue是一套前端框架，免除原生JavaScript中的DOM操作，简化书写
基于MVVM（Model-View-ViewModedl）思想，实现数据的双向绑定，将编程的关注点放在数据上

快速入门

  1. 新建HTML页面，引入Vue.js文件
  1. 在JS代码区域，创建Vue核心对象，定义数据模型
  1. 编写视图

插值表达式：形式：{{表达式}}，内容可以是：变量、三元运算符、函数调用、算术运算

常用指令

v-bind：为HTML标签绑定属性值，如设置href、css等

v-model：在表单元素上创建双向数据绑定

  • 通过文本框输入URL，点击链接跳转到指定的URL

v-on：为HTML标签绑定事件

  • 为按钮绑定事件方法

v-if、v-else-if、v-else：条件性地渲染某元素，判定为true时渲染，否则不渲染
v-show：根据条件展示某元素，区别在于切换的是display属性的值

  • 通过文本框输入年龄，判定是年轻人、中年人还是老年人

v-for：列表渲染，遍历容器的元素或者对象的属性

  • 定义数组，并渲染列表和带序号的列表

生命周期

Vue生命周期的八个阶段：每触发一个生命周期事件，会自动执行一个生命周期方法（钩子）

  1. beforeCreate：创建前
  2. Created：创建后
  3. beforeMount：挂载前
  4. Mount：挂载完成
  5. beforeUpdate：更新前
  6. updated：更新后
  7. beforeDestroy：销毁前
  8. destroyed：销毁后

mounted：挂载完成，Vue初始化成功，HTML页面渲染成功（发送请求到服务器，加载数据）
